# SmartPantryTracker

A full-stack application to track pantry items, manage inventory, and reduce food waste using smart insights.

## 🚀 Tech Stack

* Frontend: Next.js
* Backend: Node.js (Express)
* Database: MongoDB

## ✨ Features

* Add / Remove Pantry Items
* Expiry Date Tracking
* Smart Alerts for Expiring Items
* Dashboard Overview

## 📦 Installation

### Frontend

cd frontend
npm install
npm run dev

### Backend

cd backend
npm install
npm start

## 🔐 Environment Variables

Create a `.env` file in backend:
MONGO_URI=your_mongodb_connection
PORT=5000
